Guess the Number

Instructions:

  1. Guess the number between 1 and 40.
  2. You have 6 chances to guess the correct number.
  3. Click a button to guess the number.
  4. If the button number is greater than the generated number, it turns red.
  5. If the button number is less than the generated number, it turns yellow.
  6. If you guess correctly, it turns green and the game stops.
